# 敬語練習アプリ

## 概要
Spring Boot と Thymeleaf を用いて作成した、敬語表現をクイズ形式で学習できる小規模 Web アプリです。  
場面に応じて適切な敬語を選び、回答後に正誤判定・正しい答え・表現の変化・解説を確認できます。

## 主な機能
- 15問の敬語クイズ
- 回答後の正誤判定
- 正しい答えの表示
- 普通の表現 / 変化した表現 / 解説の表示
- 次の問題への遷移
- 最後の問題後に最初からやり直し可能

## 使用技術
- Java
- Spring Boot
- Thymeleaf
- HTML
- CSS
